## What I Studied

## What I Learned

A loop is used to execute a block of code multiple times.

The for loop has three parts:

1. Initialization
2. Condition
3. Increment/Decrement

Syntax:

```cpp
for(initialization; condition; increment)
{
    // code
}
```
